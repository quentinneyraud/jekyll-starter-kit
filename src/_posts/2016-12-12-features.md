#### Debug

- Define `dbg` with a context

``` javascript
const dbg = debug('app:HomePage')

dbg('Hello world')
```

- Type `localStorage.debug="app:*"` in browser console or `localStorage.debug="app:HomePage"` to only debug HomePage.js file
